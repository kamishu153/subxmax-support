## アプリについて

Sub-X Max は、Apple HealthKit のランニングデータを活用し、マラソンの目標達成をサポートするアプリです。

- 距離・ペース・VO2Max・体重の可視化
- インターバルトレーニングの自動解析
- AIコーチによる週次レビュー
- 目標レース管理とアーカイブ

## プライバシーポリシー

- 本アプリはユーザーの健康データをApple HealthKitから読み取りますが、外部サーバーへの送信は一切行いません
- すべてのデータは端末内にのみ保存されます
- 第三者AIサービスへのデータ送信

本アプリの「AIコーチ」機能では、ユーザーの同意を得た上で、
以下のトレーニングデータをGoogle LLC（Gemini API）に送信します。

【送信されるデータ】
・走行距離、ペース、走行回数
・インターバルトレーニング記録
・VO2Max、体重（平均値）
・目標レース名、目標タイム
・練習計画
・過去のコーチングコメント

【利用目的】
・AIコーチングコメントの生成のみに使用されます

■ 第三者AIサービスへのデータ送信

本アプリの以下の機能では、ユーザーの明示的な同意後にのみ、
Google LLC（Google Gemini API / https://generativelanguage.googleapis.com）に
必要最小限のデータを送信します。

【1. AIコーチ機能】
送信データ: 走行距離・ペース・走行回数・インターバル記録・
VO2Max・体重（平均値）・目標レース名・目標タイム・
練習計画・過去のコーチングコメント

【2. ニュース取得機能】
送信データ: 登録済みレース名、登録済みシューズ名

【3. レース情報自動取得機能】
送信データ: ユーザーが入力したレース名

【利用目的】
送信データは各リクエストへの応答生成のみに使用されます。
Googleのプライバシーポリシー（https://policies.google.com/privacy）
に従い処理されます。

【同意と無効化】
初回のAI機能使用時に明示的な同意画面を表示し、同意を得た上でのみ
データ送信を行います。同意後も、アプリ設定画面の「AI機能を有効にする」
トグルからいつでも無効化できます。
無効化中はデータ送信は一切行われません。
