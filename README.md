## アプリについて

Sub-X Max は、Apple HealthKit のランニングデータを活用し、マラソンの目標達成をサポートするアプリです。

- 距離・ペース・VO2Max・体重の可視化
- インターバルトレーニングの自動解析
- AIコーチによる週次レビュー
- 目標レース管理とアーカイブ

## プライバシーポリシー

- 本アプリはユーザーの健康データをApple HealthKitから読み取りますが、外部サーバーへの送信は一切行いません
- すべてのデータは端末内にのみ保存されます
- AIコーチ機能はGoogle Gemini APIを使用しますが、送信されるのは集計済みの統計データのみです
