## アプリについて

Sub-X Max は、Apple HealthKit のランニングデータを活用し、マラソンの目標達成をサポートするアプリです。

- 距離・ペース・VO2Max・体重の可視化
- インターバルトレーニングの自動解析
- AIコーチによる週次レビュー
- 目標レース管理とアーカイブ

## プライバシーポリシー

- 本アプリはユーザーの健康データをApple HealthKitから読み取りますが、外部サーバーへの送信は一切行いません
- すべてのデータは端末内にのみ保存されます
- 第三者AIサービスへのデータ送信

本アプリの「AIコーチ」機能では、ユーザーの同意を得た上で、
以下のトレーニングデータをGoogle LLC（Gemini API）に送信します。

【送信されるデータ】
・走行距離、ペース、走行回数
・インターバルトレーニング記録
・VO2Max、体重（平均値）
・目標レース名、目標タイム
・練習計画
・過去のコーチングコメント

【利用目的】
・AIコーチングコメントの生成のみに使用されます

【第三者のプライバシーポリシー】
・Google プライバシーポリシー: https://policies.google.com/privacy

ユーザーはアプリ内で初回利用時に明示的な同意を求められ、
同意しない場合はAIコーチ機能を利用できません。
