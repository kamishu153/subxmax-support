<h3>アプリについて</h3>
<p>Sub-X Max は、Apple HealthKit のランニングデータを活用し、マラソンの目標達成をサポートするアプリです。</p>
<ul>
  <li>距離・ペース・VO2Max・体重の可視化</li>
  <li>インターバルトレーニングの自動解析</li>
  <li>AIコーチによる週次レビュー</li>
  <li>目標レース管理とアーカイブ</li>
</ul>

<h2>プライバシーポリシー</h2>

<h3>1. HealthKitデータの取り扱い</h3>
<p>本アプリは、Apple HealthKitから以下のデータを<strong>読み取り専用</strong>で取得します：</p>
<ul>
  <li>ランニングワークアウト（距離・時間・ペース・心拍数等）</li>
  <li>VO2Max</li>
  <li>体重・体脂肪率</li>
</ul>
<p>取得したデータは<strong>お使いの端末内にのみ</strong>保存され、AI機能を利用しない限り外部送信されることはありません。</p>

<h3>2. 第三者AIサービス（Google Gemini API）へのデータ送信</h3>
<p>本アプリには、Google LLCが提供するGoogle Gemini APIを使用する<strong>AI機能</strong>が含まれます。これらの機能は<strong>デフォルトでは無効</strong>になっており、ユーザーがアプリ設定画面の「AI機能を有効にする」トグルをONにし、同意ダイアログで明示的に同意した場合のみ利用可能になります。</p>

<p><strong>【送信先】</strong><br>
Google LLC（Google Gemini API）<br>
エンドポイント: https://generativelanguage.googleapis.com</p>

<p><strong>【AI機能①：AIコーチ機能】</strong><br>
送信されるデータ：</p>
<ul>
  <li>走行距離、ペース、走行回数</li>
  <li>インターバルトレーニング記録</li>
  <li>VO2Max、体重（平均値）</li>
  <li>目標レース名、目標タイム</li>
  <li>練習計画</li>
  <li>過去のコーチングコメント</li>
</ul>
<p>利用目的：コーチングコメントの生成</p>

<p><strong>【AI機能②：ニュース取得機能】</strong><br>
送信されるデータ：</p>
<ul>
  <li>ユーザーが登録したレース名</li>
  <li>ユーザーが登録したシューズ名</li>
</ul>
<p>利用目的：関連ニュース記事の検索・取得</p>

<p><strong>【AI機能③：レース情報自動取得機能】</strong><br>
送信されるデータ：</p>
<ul>
  <li>ユーザーが入力したレース名</li>
</ul>
<p>利用目的：大会公式URL・開催日・エントリー開始日の取得</p>

<h3>3. 同意と無効化</h3>
<ul>
  <li>初回のAI機能使用時に明示的な同意画面を表示します</li>
  <li>同意しない場合、AI機能は利用できません</li>
  <li>同意後も、アプリ設定画面の「AI機能を有効にする」トグルから<strong>いつでも無効化</strong>できます</li>
  <li>無効化中は、すべてのAI機能（3種類とも）でデータ送信は一切行われません</li>
</ul>
